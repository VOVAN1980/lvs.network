LVS — Legal Center

Central hub for licensing, governance, security and policy documents of LVS — Living Value System.

Website Policies

These documents apply to visitors of lvs.network and all public demo pages.

  • Terms of Use — usage rules, liability and disclaimers for the website and demo tools.
  • Privacy Policy — GDPR-aligned description of what data is (not) collected, cookies, logging and user rights.

Licensing

LVS uses a composite licensing model: open research documentation plus a restricted core implementation.

  • LICENSE — combined licensing for documentation and core technology.
  • License Overview — human-readable summary of the composite model.
  • LVS Core Technology License — restricted license for the protocol engine and node/runtime implementations.

Governance & Contribution

These documents describe how changes to LVS are proposed, reviewed and accepted, and how contributors are expected to work with the project.

  • Governance — roles, LIP process, decision-making and versioning rules.
  • Contributing — contribution workflow, technical requirements and documentation rules.
  • Code of Conduct — required standards of behavior for all participants.

Security & Trademarks

LVS is a security-sensitive research project with protected branding and a strict private disclosure process.

  • Security Policy — how to responsibly report vulnerabilities and what is out of scope.
  • Trademark Policy — rules for using the “LVS”, “Living Value System”, “LVS Core” and “LVS Network” names and logos.

Scope

The documents listed here cover the LVS research repositories, the LVS Core implementation, and the public website lvs.network. They may be updated as the project evolves (e.g., when the LVS Foundation is formally established or when Testnet/Mainnet go live).

For the latest protocol specifications, whitepaper and technical architecture, please refer to the official LVS documentation set linked from the main page.
