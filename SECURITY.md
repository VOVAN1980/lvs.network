# Security Policy

LVS Network is a critical infrastructure project.  
Security is a top priority.  
We welcome responsible disclosures from security researchers.

---

## Supported Components

This repository includes:

- Documentation  
- Website assets  
- High-level specifications  

No core protocol code is included here.  
However, vulnerabilities related to documentation accuracy, website integrity, or ecosystem safety are still important.

---

## Reporting a Vulnerability

Please report all security issues privately.

**Email:** security@lvs.network  
**PGP Key:** (to be added)  
**Type:** Responsible Disclosure  

Do NOT:

- publish vulnerabilities publicly  
- create GitHub issues about security problems  
- discuss vulnerabilities in social networks or chats  

We respond within **72 hours** and provide status updates within **7 days**.

---

## Scope

In scope:

- lvs.network website  
- documentation inconsistencies that may lead to protocol misuse  
- branding impersonation  
- infrastructure misconfiguration  

Out of scope:

- speculative attacks on unreleased protocol code  
- theoretical cryptographic research without a practical exploit  
- phishing not controlled by LVS
