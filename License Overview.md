## 2. Core Implementation (Restricted)

The LVS protocol implementation is **NOT open-source**.

Covered by the **LVS Core Technology License**:

- src/
- consensus logic
- validator logic
- networking layer
- runtime modules
- any future node implementation

Purpose:  
Controlled development, prevention of unauthorized forks or premature networks.

Allowed:
- local research
- testing
- reading the source
- submitting improvements

Not allowed:
- public deployment
- financial/token use
- commercialization
- re-licensing
- derivative protocols or networks

Full restrictions: see `LICENSE_TECHNOLOGY.md`.
